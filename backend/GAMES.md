## Catatan Teknis

- Semua game HTML5 Canvas / DOM, berjalan di WebView
- Bundle: zip file (index.html + game.js + assets), di-upload via Django admin
- Timer & coin deduction di native layer (anti-cheat)
- Game communicate via postMessage API
- Game bisa pakai kata-kata dari buku yang sudah dibaca anak (context-aware)
- Offline playable setelah download, tapi start session perlu online
- Admin bisa preview game di iframe sebelum publish
